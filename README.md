# Bin2Dec

> 💡 This project was built as part of the [App Ideas Collection](https://github.com/florinpop17/app-ideas) by Florin Pop.

A simple binary-to-decimal converter that helps developers practice and understand binary (base 2) mathematics — the number system all digital computers are built on.

## Features

- Enter up to 8 binary digits (0s and 1s) in a single input field
- Instantly see the decimal (base 10) equivalent
- Get notified with an error message if invalid characters are entered

## Constraints

This project was built with the following intentional constraints as part of the challenge:

- No arrays used to store binary digits
- Decimal conversion is calculated using a single mathematical function (power of 2 via exponentiation)

## How to Use

1. Type a binary number (up to 8 digits) into the input field — e.g. `1101`
2. The decimal equivalent will be displayed in the output field
3. If you enter anything other than `0` or `1`, an error message will appear
